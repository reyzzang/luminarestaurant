# Lumina - Premium Restaurant Website

A modern, responsive, and luxurious restaurant website built with pure HTML, CSS, and Vanilla JavaScript. 

## Features
- **Modular Architecture:** Reusable HTML components (Navbar, Footer) loaded via JavaScript.
- **i18n Support:** Dynamic language switching (English, Georgian, Russian).
- **Responsive Design:** Mobile-first approach with custom CSS variables.
- **Performance:** Intersection Observer for lazy scroll animations.
- **SEO Ready:** Complete with sitemap, robots.txt, and schema.org structured data.

## Setup
Due to the modular architecture using the Fetch API for components and languages, this project must be run on a local server.
1. Install a local server (e.g., "Live Server" extension in VS Code).
2. Open `index.html` with the local server.
